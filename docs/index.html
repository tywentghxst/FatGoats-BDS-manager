Operation Setup Desk

Deploy In Less Than 5 Minutes

Choose your hosting environment stack framework to unlock easy step-by-step commands.

1
Install Node.js

Download and install the official Node.js (v18 or newer LTS) runtime from nodejs.org.

2
Download App & Launch Setup Builder

Extract the repository files locally. Double click the automated fast setup builder script:

build.bat
3
Launch Manager Service

Spin up node systems completely preconfigured in one single task run:

start-windows.bat
4
Manage & Install

Open your browser to http://localhost:3000. Access the panel with your local authentication secrets!

1
Generate Isolated Docker Container Image

Compile frontend segments and pack backend configurations into isolated images with:

docker build -t bedrock-manager .
2
Compose Setup Deployment Orchestrations

Start using Docker Compose to handle ports and volume folder bindings. Run:

docker compose up -d
3
Verify Port Setup Ingress Rules

Confirm that port 3000 (TCP) is maps correctly for Admin access, and UDP port 19132 (UDP) is fully authorized on your physical engine routers!
